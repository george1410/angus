What is the 3D Snow Globe?

This small project is a web-based 3D environment, written in JavaScript. I found a tutorial on Enlight for building a 3D Snow Globe, so I decided to follow it. This tutorial introduced me to the Three.js library, which is used for generating web-based 3D envrionments.

This tutorial showed me some of the basics of the Three.js library, where it had me build a basic 3D environment that involved randomly generated snowflakes that moved across the screen. All of the logic was explained clearly, and links to the relevent documentation were available for me to read whenever a new method was introduced. Once I had completed the tutorial, I was free to change different parts of the code to see how it impacted the environment. I found this an effective way of understanding the code better.
