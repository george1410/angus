What is the Random Number Guessing Game?

This random number guessing game is a small Python project of mine. The aim was to create a procedural application for my BTEC Level 3 Coursework, and I deciced to create this. The game is written entirely in Python, and it involves a user attempting to guess a randomly generated number between 1 and 100. The user only gets 5 guesses, so it's a tough game, but they are helped along by hints on whether to guess higher or lower.

The development of this application is fully documented, and I will put this documentation up on my Google Docs. Documentation for this application includes things such as devlopment planning and testing.
